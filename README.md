# ⚡ SetUp | Fitness Performance App

**SetUp** est une application mobile de fitness haute performance conçue pour offrir une expérience d'entraînement fluide, esthétique et motivante. Développée avec **React Native** et **Expo**, l'application mise sur un design "Dark UI" moderne avec des accents **Electric Green** (#4ADE80).

---

## ✨ Fonctionnalités

* **⚡ Interface Électrique :** Design immersif optimisé pour la concentration et la performance.
* **📊 Statistiques Avancées :** Suivi en temps réel des séances, calories brûlées et records personnels.
* **🎨 Thémage Dynamique :** Personnalisation complète de la couleur d'accentuation (6 thèmes inclus).
* **🔔 Rappels Intelligents :** Système de notifications quotidiennes pour ne jamais rater une séance.
* **📳 Retour Haptique :** Expérience tactile premium intégrée à chaque interaction.

---

## 🛠️ Stack Technique

* **Framework :** React Native (Expo SDK)
* **Langage :** JavaScript / ES6
* **Navigation :** Expo Router / Navigation
* **Assets :** Expo Vector Icons (Ionicons & MaterialCommunityIcons)
* **Logic :** React Context API (pour le thémage et les réglages haptiques)

---

## 🚀 Installation & Lancement

Pour faire tourner **SetUp** sur ton propre téléphone ou simulateur, suis ces étapes simples :

### 1. Cloner le projet
Ouvre ton terminal et récupère le code :

```bash
git clone [https://github.com/quentynustolle-dot/SetUp.git](https://github.com/quentynustolle-dot/SetUp.git)
cd SetUp
```
### 2. Installer les dépendances

Assurez-vous d'avoir Node.js installé, puis lance :
```Bash

npm install
```
### 3. Lancer l'environnement Expo

Démarre le serveur de développement :
Bash
```
npx expo start
```
### 4. Visualiser l'application

  Sur Mobile (Recommandé) : Installe l'application Expo Go (disponible sur l'App Store et Play Store). Scanne le QR Code qui s'affiche dans ton terminal.

  Sur l'ordi : Appuie sur w pour lancer la version Web ou a pour l'émulateur Android.
